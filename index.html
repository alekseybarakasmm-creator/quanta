Наш API строго типизирован и сопровождается детальной документацией. Начните принимать платежи за несколько строк кода.

const quanta = require('quanta')('sk_test_...');

// Создание платежной сессии
const intent = await quanta.payments.create({
  amount: 2000, // в копейках
  currency: 'rub',
  automatic_methods: { enabled: true },
  receipt_email: 'client@mail.ru',
});

console.log(intent.client_secret);

Пользовательское соглашение и Оферта

Настоящее Соглашение (далее — Соглашение) регулирует отношения между платформой Quanta (далее — Платформа) и пользователем сети Интернет (далее — Пользователь), возникающие при использовании API и сервисов Платформы.

1. Общие положения

Использование сервисов Платформы регулируется настоящим Соглашением. Платформа оставляет за собой право вносить изменения в Соглашение без предварительного уведомления.

2. Политика обработки данных (API)

Все транзакционные данные обрабатываются в соответствии со стандартом PCI DSS Level 1. Платформа не хранит CVV/CVC коды банковских карт пользователей.

Важно для разработчиков

При интеграции Quanta Elements, вы обязаны разместить ссылку на Политику конфиденциальности на странице оплаты вашего сайта (согласно 152-ФЗ).
